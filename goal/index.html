Вам будет предоставлено

Статистические данные более чем для 300 футбольных лиг всего мира. 43 статистических параметра для каждого матча. Возможность сортировки данных как по каждому статистическому параметру в отдельности, так и по нескольким параметрам одновременно. Экспорт в Excel всех статистических данных для их более детального и глубокого анализа.

Статистические параметры:

  • Х_ГЗ - общее количество голов, забитых хозяевами, когда они играли дома
  • Х_ГП - общее количество голов, пропущенных хозяевами, когда они играли дома
  • Х_ГЗСМ - количество голов, забиваемых хозяевами в среднем за матч, когда они играли дома
  • Х_ГПСМ - количество голов, пропускаемых хозяевами в среднем за матч, когда они играли дома
  • Х_ГЗ+ГПСМ - общее количество голов, забитых + пропущенных в среднем за матч домашней команды при игре дома
  • Х_ТБ1,5% - % матчей с 2-мя и более голами, когда хозяева играли дома
  • Х_ТБ2,5% - % матчей с 3-мя и более голами, когда хозяева играли дома

Больше не нужно перелопачивать кучу сайтов с футбольной статистикой, ВСЁ собрано в ОДНОЙ таблице.
